Using Figma To Produce a
Prototype For Your Fitness App

Written By Palvisha Shoaib

Introduction:

Figma is one of the most prominent web-based graphic editing, prototyping and user interface design
applications. Figma is different as compared to the other design softwares, as it is very easily accessible,
and has a collaboration feature which lets you collaborate with other team members.
Figma can be used to do all kinds of design work, from graphic design to making high-fidelity screens
and designing websites, mobile app interfaces, and for making prototypes for your app or website.

In this tutorial we are going to use Figma Design tool to create 2 views for our fitness app!
A get started view and a login view. This tutorial is for Figma designers who are beginners
who want to learn some basics.

The idea of our fitness app is very simple. The ‘Get Started’ page would have a background image,
with a logo and a text including a ‘Get Started’ button. The login view is so that users should
be able to login easily and fast.

Prerequisites:

To complete this tutorial, you will need:

  • Access to a computer.
  • Need an account on Figma which you can easily make by going to the
    Figma website, clicking “Sign up”, and then entering your details.

So let's dive in and create our first project!

1. Creating a new file.

To start with this tutorial, you need to create an empty file. Once we have created our ‘New File’,
we can go ahead and get started by developing the frames for our fitness app.

2. Developing design frames.

In the top left of the tool bar, we see that there is a ‘Frame’ tool.
Once we select this, we see that there are a lot of options from which we
could use any design frame we like. I’m going to use iPhone 11 pro, but you can create
the prototype of this app with any design frame you prefer.

3. Copying frames.

Once we have our frames selected we copy that frame one more time, so we have
them all side by side.

4. Setting the background image, logo and logo text.

First we are going to select a background image to our ‘Get started’ view,
and the image that I’m going to be using is from Unsplash, and I’m also going to add a logo
image here which is an SVG that I picked from Flaticon.

Once you have found your image, and logo, you can drag and drop to the Figma page and
crop it or resize it so that it could fit the screen. You can also edit the image the
way you like, here I added a drop shadow at the top, to make the image look more faded.
You can add the drop shadow by using the ‘Fill’ option on the side, and can increase or
decrease the shadow while also setting the color. I did a linear drop shadow with black 100%.
Now we’ve added our title of the app, and the logo SVG and the title of the app. You can add
the title of the app by using the ‘T’ option on the left side of the tool bar. I’m using Nunito
bold 36px font for the app logo text. Feel free to play about with different fonts or even a logo.

5. Creating the ‘Get Started’ button.

Next up is our “Get Started” button, for that we need to create a rectangle,
adjust its corner radius which you can do on the right side of the menu, I have my corner
radius set to 25 but you can do anything you prefer. Then, we will change the fill color to
black so it could match our theme.

I increased the corner radius and added text, I’m using Nunito bold 18px.
After I added the text, I searched for an arrow icon that I got from Flaticon, copied it,
and pasted it right next to the ‘Get Started’ text.

Now for our second view, the ‘Login View’.

1. Adding the logo, logo text, and a statement.

First, we are going to set the background color of the frame to black
which could be done by using the ‘Fill’ tool to add color. I added a similar drop shadow
that I did before on the previous screen. I added a linear white drop shadow here with 40%.
Then we’ll drag and drop our SVG logo, resize it so it fits the screen, and then place it at
the top with the same logo text we had on the first page. I added a logo statement at the bottom
that matches our app, but you can add any statement you prefer.

2. Adding input fields to the screen.

Next up we are adding input fields to the view, for that we need to draw
two squares under the statement and then adjust the corner radius from the menu to the right side.
The corner radius I have over here is 20, but you can choose any. Once we have them drawn, we can
fill the rectangles with white, and add the ‘username’ ‘password’ field with a lighter grey color
and place the text into the rectangles.

Then, we create a ‘Login’ button which is slightly smaller than the other two, and fill it with
white as well. Next, I put ‘Login’ text in black using the same font.

Lastly, I placed some more text under the ‘Login’ button, and added ‘Forgot your password’
so that users could reset their password if they need to.

Now we have 2 views that we can use to show our app idea or collaborate with other team members!

Figma is such an amazing and easy to use designing and prototyping tool that could be used by
graphic designers, UI, UX, software developer or web designers. Now that we’re done with our 3 views,
we could use our app idea and share it with other team members, and can collaborate as well by using
the ‘Share’ button on top right of your project.

I hope you found this tutorial useful, and learned a few basics from this!
